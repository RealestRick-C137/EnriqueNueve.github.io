1. Intro

  A key task in machine learning is density estimation. Density estimation is the process of estimating from a set of samples the underlying distribution of the data. Density estimation can provide insight into the probability of a sample but also can be used in more direct applications such as classification, denoising, missing input imputation, and data synthesis. In this post, a density estimation method by the name of MADE, Masked Autoencoder for Density Estimation is examined along with a code implementation in Tf2.
